# Karabiner-Elements

## Programming layer

Tap `ä` to type `ä`. Hold `ä` to activate the programming layer.

```text
          e → =    r → $    t → +    z → *    u → &    i → !    o → ?

a → ^    s → {    d → (    f → [    g → /    h → \    j → ]    k → )    l → }

                            v → '    b → |    n → "    m → `
```

The `z` mapping uses Karabiner's `y` key code because the German layout swaps the physical `Y` and `Z` positions.

## Number and navigation layer

Tap `ö` to type `ö`. Hold `ö` to activate the number and navigation layer.

```text
Caps Lock → 0

                                          z → Left    u → Down    i → Up    o → Right

a → 1    s → 2    d → 3    f → 4    g → 5    h → 6    j → 7    k → 8    l → 9
```

Caps Lock continues to produce Escape outside the number layer.

The `z` mapping uses Karabiner's `y` key code because the German layout swaps the physical `Y` and `Z` positions.

## Navigation layer

Tap `-` to type `-`. Hold `-` to activate the navigation layer.

```text
a → Shift    s → Control    d → Option    f → Command           g/h → Left    j → Down    k → Up    l → Right
```

Modifiers can be combined with the navigation keys. For example, hold `-`, then hold `f` and press `j` to send Command+Down Arrow.

The physical left Shift, Control, Option, and Command keys work as alternatives to `a`, `s`, `d`, and `f`.

NOTE: `h` does not work with modifiers somehow. Use `g` instead then.

## FnMate layer

Hold `fn` for the former Hammerspoon FnMate mappings.

```text
h → Left    j → Down    k → Up    l → Right

space → Return    - → Backspace

z/o → vertical scroll    u/i → horizontal scroll

, → left click    . → right click
```

The `z` mapping uses Karabiner's `y` key code because the German layout swaps the physical `Y` and `Z` positions.
