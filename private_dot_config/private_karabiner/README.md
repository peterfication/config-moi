# Karabiner-Elements

## Programming layer

Tap `ä` to type `ä`. Hold `ä` to activate the programming layer.

```text
          e → =    r → $    t → +    z → *    u → &    i → ?

a → ^    s → {    d → (    f → [    g → /    h → \    j → ]    k → )    l → }

                            v → '    b → |    n → "    m → `
```

The `z` mapping uses Karabiner's `y` key code because the German layout swaps the
physical `Y` and `Z` positions.
